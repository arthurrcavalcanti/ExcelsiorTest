# ExcelsiorTest

## Segue o desafio:
A área do Financeiro da Excelsior está efetuando diversos cálculos de forma manual.

### História:
Eu quero uma API no qual eu possa efetuar as 4 operações matemáticas básicas (Soma, Subtração, Multiplicação e Divisão). Essas operações deverão ser aplicadas apenas em 2 números por vez.

### Cenário de teste 1
Eu quero enviar um json para o endpoint (POST) da API chamado SOMA
- NumberOne = 10
- NumberTwo = 10
- Result: 20

### Cenário de teste 2
Eu quero enviar um json para o endpoint (POST) da API chamado SUBTRAÇÃO
- NumberOne = 100
- NumberTwo = 20
- Result: 80

### Cenário de teste 3
Eu quero enviar um json para o endpoint (POST) da API chamado MULTIPLICAÇÃO
- NumberOne = 5
- NumberTwo = 10
- Result: 50

### Cenário de teste 4
Eu quero enviar um json para o endpoint (POST) da API chamado DIVISÃO
- NumberOne = 100
- NumberTwo = 10
- Result: 10
